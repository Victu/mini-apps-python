# 💱 Aplicativo de Câmbio 💱

Um aplicativo simples com Interface Gráfica de Usuário (GUI) que desenvolvi em Python que busca e exibe, em tempo real, as cotações do Dólar (USD), Euro (EUR) e Bitcoin (BTC) em relação ao Real Brasileiro (BRL).

## 🛠️ Funcionalidades

* Interface amigável, minimalista e direta ao ponto construída com `Tkinter`.
* Busca as taxas de câmbio em tempo real com apenas um clique.
* Consome dados confiáveis da API pública de moedas [AwesomeAPI](https://docs.awesomeapi.com.br/api-de-moedas).
* Exibição formatada e alinhada dos valores.
